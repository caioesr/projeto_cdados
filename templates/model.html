Modelo preditivo

Neste página você poderá usar o modelo para prever se alguma transação bancária é fraude ou não. Tudo que você precisa fazer é submeter um arquivo to tipo csv para que nosso preditor faça o trabalho de classifica-lo como fraude ou não. Você pode enviar diversos casos (como uma tabela) de transações bancárias, o modelo irá retornar a classificação de todas elas.
